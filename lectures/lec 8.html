  • This is a paragraph with relative units of 1rem. It depends of font size of html
    If the root element has a font size of 16 pixels, 1rem would equal 16 pixels

  • This is another paragraph with relative units of 1vw.(Browser Width)
    vw is a unit of measurement in CSS that represents 1% of the viewport width. If the viewport is 1200 pixels wide, 100vw would equal 1200 pixels.

  • This is yet another paragraph with relative units of 1vh.(Browser height)
    vh is a unit of measurement in CSS that represents 1% of the viewport height. If the viewport is 800 pixels tall, 100vh would equal 800 pixels.

  • This is yet another paragraph with relative units of 100%.

    If an element has a font size of 16 pixels, 1em would equal 16 pixels. If the font size of the element is changed, the size of 1em will change proportionally.
